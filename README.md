# Fast File
![Logo](logo.png)
---
Fast File Server is a file server system based on MongoDB.

It's using some very popular technology like:
- MongoDB
- SpringBoot2.0.4
- SpringWebflux
- MongodbReactive

## Features
- [ ] File CRUD
- [ ] File upload and download
- [ ] Image compression
- [ ] Image style（stream）
- [ ] Systems Manager (VUE/React)
